# 🐰 염버니봇 (ScorbunnyBot)

친구들과 함께 사용하는 디스코드 봇. 음악 재생, 발로란트 상점 조회, 메이플스토리 정보 조회를 지원합니다.

---

## 🛠️ 기술 스택

| 항목 | 내용 |
|------|------|
| **언어** | Python 3.11+ |
| **Discord** | discord.py (with voice) |
| **음악** | yt-dlp, FFmpeg, PyNaCl |
| **HTTP** | aiohttp, certifi |
| **DB** | SQLite (aiosqlite) |
| **암호화** | cryptography (Fernet) |
| **크롤링** | Playwright, BeautifulSoup4 |
| **배포** | Docker + AWS EC2 |

---

## ✨ 기능

### 🎵 음악

YouTube URL 또는 검색어로 음악을 재생합니다. 재생 기록은 DB에 저장되고 자동재생 및 분기별 RECAP에 활용됩니다.

| 커맨드 | 설명 |
|--------|------|
| `/입장` | 음성 채널에 입장 |
| `/재생 [검색어/URL]` | URL 또는 검색어로 음악 재생 |
| `/스킵` | 현재 곡 건너뛰기 |
| `/일시정지` | 음악 일시정지 |
| `/다시재생` | 일시정지된 음악 재개 |
| `/대기열` | 현재 대기열 확인 |
| `/삭제 [번호]` | 대기열에서 특정 곡 삭제 |
| `/반복` | 대기열 전체 반복 |
| `/한곡반복` | 현재 곡 반복 |
| `/음량 [1~100]` | 음량 조절 |
| `/자동재생` | 자동재생 ON/OFF (재생 기록 30곡 이상 필요) |
| `/히스토리` | 최근 재생한 곡 목록 확인 |
| `/리캡 [분기]` | 분기별 TOP 10 플레이리스트를 대기열에 추가 |
| `/나가` | 음성 채널에서 퇴장 |

**자동재생:** 대기열이 비었을 때 이전 재생 기록에서 랜덤으로 곡을 선택해 재생합니다. 최근 10곡은 제외되며, 재생 기록이 30곡 이상 쌓여야 활성화됩니다.

**RECAP:** 매 분기(1/4/7/10월 1일 자정)에 해당 분기의 재생 통계와 TOP 5를 지정 채널에 자동 발송합니다. 10분 이상의 곡은 기록에서 제외됩니다.

### 🎮 발로란트

Riot 공식 OAuth를 통해 발급된 access_token으로 개인 상점을 조회합니다. 토큰은 Fernet으로 암호화하여 DB에 저장됩니다.

| 커맨드 | 설명 |
|--------|------|
| `/로그인` | Riot 계정 연동 안내 |
| `/인증 [URL]` | 로그인 후 리다이렉트된 URL로 인증 |
| `/상점` | 오늘의 발로란트 스킨 상점 확인 |
| `/야시장` | 야시장 할인 목록 확인 |
| `/로그아웃` | 연동된 계정 정보 삭제 |

> 토큰 유효 시간은 1시간입니다. 만료 시 로그아웃 후 재로그인이 필요합니다.

---

### 🍁 메이플스토리

Nexon Open API를 이용해 캐릭터 정보를 조회하고, 매주 썬데이 메이플 공지를 자동으로 수집해 지정 채널에 전송합니다.

| 커맨드 | 설명 |
|--------|------|
| `/메이플 [닉네임]` | 캐릭터 기본 정보 및 장비 조회 (관리자 전용) |
| `/썬데이` | 이번 주 썬데이 메이플 정보 조회 (관리자 전용) |

**썬데이 메이플 자동 알림:** 매주 금요일 오전 10:10 ~ 12:10 KST 사이에 30분마다 공식 홈페이지를 체크하여 새 공지가 올라오면 지정 채널에 자동 전송합니다. (일요일 자정 백업 체크 포함)

## ⚙️ 관리자 커맨드 (Prefix: `!`)

| 커맨드 | 설명 |
|--------|------|
| `!봇채널설정 #채널` | RECAP 발송 채널 설정 |
| `!메이플알림설정 #채널 [@역할]` | 썬데이 메이플 알림 채널 설정 |
| `!메이플알림해제` | 썬데이 메이플 알림 해제 |
| `!썬데이테스트` | 알림 채널에 즉시 전송 테스트 |
| `!recap테스트` | RECAP 즉시 발송 테스트 |
| `!recap미리보기` | 현재 분기 RECAP 미리 확인 |
| `!음악커맨드` | 음악 관리자 커맨드 가이드 |
| `!메이플커맨드` | 메이플 관리자 커맨드 가이드 |

## 🗃️ DB 구조 (SQLite)

| 테이블 | 용도 |
|--------|------|
| `users` | 발로란트 유저 인증 정보 (토큰 암호화 저장) |
| `play_history` | 서버별 재생 기록 (자동재생용, 곡당 1행) |
| `recap_history` | 서버별 재생 기록 (RECAP용, 재생마다 1행) |
| `guild_settings` | 서버별 RECAP 채널 설정 |
| `sunday_channels` | 썬데이 메이플 알림 채널 및 역할 |
| `sunday_state` | 마지막으로 전송한 썬데이 URL (중복 전송 방지) |
| `login_sessions` | 임시 로그인 세션 |

## 🚀 실행 방법

### 1. 환경변수 설정

`.env` 파일을 생성하고 아래 항목을 입력합니다.

```
DISCORD_TOKEN=your_discord_bot_token
OWNER_ID=your_discord_user_id
ENCRYPTION_KEY=your_fernet_key
NEXON_API_KEY=your_nexon_api_key
```

Fernet 키 생성:
```python
from cryptography.fernet import Fernet
print(Fernet.generate_key().decode())
```

### 2. 의존성 설치

```bash
pip install -r requirements.txt
```

FFmpeg는 별도로 설치가 필요합니다.

```bash
# Windows
winget install ffmpeg

# Ubuntu
sudo apt install ffmpeg
```

### 3. 실행

```bash
python main.py
```
